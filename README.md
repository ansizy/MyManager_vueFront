# 瑟瑟本地资源管理器----前端
## 基本功能:
这是一个网页版管理本地的资源的管理器，更好的查看本地的资源。 
目前将所有资源大致分为四种类型:
 + 电影
 + 漫画
 + 社交媒体
 + 杂项 TODO
## 使用指南
### 电影
1. 采用刮削器刮削好电影。
2. 添加目录，类似于"H:\other\学习资料\#整理完成"
### 漫画
1. 需要先手动整理漫画文件夹
2. 将父文件夹加入地址，类似于"H:\other\学习资料\#整理完成"
### 社交媒体
和漫画类似
1. 需要先手动整理社交媒体文件夹
2. 将父文件夹加入地址
